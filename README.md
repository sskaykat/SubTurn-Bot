## SubTurn-Bot
---
- 一个订阅转换 telegram bot，把订阅连接转换成 clash meta 格式
- 你也可以自己修改配置文件 `config.yaml` 预设想要的分流规则
- 自建 `bot` 修改 `token.txt` 中的 `BOT_TOKEN`
- 默认规则预览：
